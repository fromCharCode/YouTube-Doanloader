=== fromCharCode ===

> Python wird benötigt: https://www.python.org/downloads/ <

Zum Ausführen bitte die downloader.bat starten, 
oder per CMD zum Verzeichnis navigieren und "python run.py" eingeben

Der YouTube zu MP3 Konverter benutzt die youtube_dl lib: https://github.com/rg3/youtube-dl
Sowie FFmpeg zum konvertieren: http://www.ffmpeg.org/

--------------------

Mit der Option 1 kann man einen Link eingeben, welcher aber in keiner privaten Liste sein darf.
Der Link selber darf aber eine Liste beinhalten, diese wird dann vollständig gedownloaded.

Mit der Option 2 kann man sämtliche Lieder aus einer Textdatei gelesen, allerdings müssen die Links
in jeweils einer neuen Zeile stehen. Als Beispiel dient die links.txt

====================


ENGLISH:

> Python is required: https://www.python.org/downloads/ <

Please run the "downloader.bat" file, or navigate to this directory via Comman Line
and insert "python run.py" to run the file.

The YouTube to MP3 converter uses the "youtube_dl" libary: https://github.com/rg3/youtube-dl
as well as FFmpeg to convert the .webm to .mp3: http://www.ffmpeg.org/

--------------------

With the first option besides the choice whether you use mp3 or wav is if you want to insert
a link. This link may contain a list which will be donwloaded completely if each video is not private or
deleted.

With the second option you can read one or multiple links from a file. The easiest way to access a file
is to put a <name>.txt file in the same directory, otherwise you'll have to keep in mind which path the file uses

====================
